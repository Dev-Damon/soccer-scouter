## 1. 현재 상태 (구현 완료)
- 일정 / 조별(실시간 순위표·3위경쟁) / 나라·선수·경기 분석 / 검색·선수 랭킹.
- **ESPN 공개 API**(키X·CORS O): 라이브 점수, 조별 순위, 역대 상대전적(H2H).
- **뉴스 자동수집**: GitHub Actions 4시간 크론 + 구글뉴스 RSS(48개국) → news.json. 날짜별 나라 뉴스.
- **댓글(Supabase)**: 좋아요/싫어요·정렬·대댓글(@태그)·신고(더보기 메뉴)·300자 + 마이페이지(닉네임·내댓글·태그된댓글) + 관리자 페이지(신고/전체/강제삭제/차단/숨김해제).
- **어뷰징·봇 방어(서버 트리거, 우회불가)**: 레이트리밋·도배·링크 전면차단·스팸/성인 키워드·교차도배·유저 밴·신고 3회 자동숨김.
- **선수 평점(⭐1~5, player_ratings)** + **선수 랭킹**(검색탭: OVR/유저평점/가치/유명도 정렬, 포지션필터, 국기카드).
- 메인 히어로(위트 문구 3초 로테이션 + 자동 D-day) · 소셜 로그인(구글) · 약관/방침 · 앱 로고.

## 2. 게시판(커뮤니티) 기획 ★ 다음 예정
목적: 선수/경기별 댓글과 **별개로**, 자유롭게 글(스레드) 올리고 토론하는 공간.
- **카테고리**: 자유 / 경기예측 / 응원·드립 / 정보공유
- **글 구조**: 제목 + 본문 + 작성자(닉네임) + 시간 + 좋아요 + 조회수 + 댓글(스레드)
- **DB**: `board_posts`(id, category, title, body, user_id, name, created_at, likes, views, hidden, reply_to_user). 글의 댓글은 **기존 comments 재사용**(thread_key = `post:{글id}`) → 좋아요·대댓글·신고 그대로 동작.
- **UI**: 게시판 진입(하단탭 추가 또는 MY 근처) → 카테고리 → 글 목록(최신순/인기순) → 글 상세(본문 + 댓글) → 글쓰기(+ 버튼, 로그인 필요)
- **방어**: 기존 어뷰징/봇 트리거·필터·신고·밴 전부 재사용(글 본문도 동일 적용)
- **관리자**: 글 강제삭제·숨김(관리자 페이지 확장)
- **단계**: 1차(자유게시판 + 글쓰기 + 댓글) → 2차(카테고리·인기정렬·조회수·이미지)

## 3. 실시간 경기정보 기획 (ESPN — 조사완료, 구현 가능)
- **데이터원**: ESPN `summary?event=` (무료·키X·CORS O·정적OK). 2022 결승으로 검증됨.
  - 라인업: 양팀 선발11 + 후보 + 포메이션(예 4-3-3)
  - keyEvents: 골 · 옐로/레드카드 · 교체 · 페널티 (분 + 선수명)
- **UI(경기 페이지)**: 라이브 점수(구현됨) + 실제 라인업(예상→실제) + 이벤트 타임라인
- **폴링**: 라이브 중 1분 간격(기존 fetchLive 모듈 확장). 백엔드 불필요.

## 4. 경기 평점 · MVP 기획 (선수 평점 확장 — 황윤진 제안)
- 현재: 선수 '종합' ⭐평점 구현됨(player_ratings).
- 확장: **경기별** 선수 평점 → 경기당 MVP → 최다 MVP = "킥톡이 뽑은 2026 월드컵 최고의 선수".
- DB: `match_ratings`(match_id, player_id, user_id, score) + 경기·선수 집계 뷰.

## 5. 외부 설정 대기 (사용자 액션 필요)
- 카카오 로그인: REST API 키 + Client Secret (받으면 Supabase에 설정)
- 구글 도메인 인증: Search Console TXT(호스트 `@`) → OAuth 로고/브랜딩
- 선수 등번호: 나머지 43개국(5개국 완료) — 병렬 에이전트로 수집

## 6. 로드맵 (우선순위 제안)
1. 게시판 1차 (자유게시판 + 글쓰기 + 댓글)
2. 실시간 경기정보 (실제 라인업 + 이벤트 타임라인)
3. 경기 평점 · MVP
4. 카카오 로그인 활성
5. 등번호 전팀 + 게시판 2차(카테고리·인기)
