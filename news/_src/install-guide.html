킥톡 앱 설치·사용법 — 홈 화면에 추가하고 월드컵 100% 즐기기

킥톡은 설치 없이 웹에서 바로 쓸 수 있지만, 홈 화면에 추가하면 앱처럼 주소창 없이 한 번에 실행돼 훨씬 편하다. 별도 앱스토어 다운로드도, 비용도 없다. 스마트폰에 킥톡을 추가하는 법과 핵심 기능 활용법을 정리했다.

왜 홈 화면에 추가할까

  • 아이콘을 눌러 앱처럼 바로 실행 — 매번 주소를 칠 필요가 없다.
  • 주소창·탭 없이 전체 화면으로 경기 정보에 집중.
  • 설치·결제 없이 모든 기능 무료.

아이폰(iPhone)에서 추가하기

아이폰은 Safari 브라우저에서 추가해야 한다.

  • ① 킥톡(kicktalk.xyz)을 Safari로 연다. (카카오톡·인스타 등 인앱 브라우저로 열렸다면, 먼저 'Safari로 열기'를 눌러 옮긴다.)
  • ② 하단의 공유 버튼 ⬆️을 누른다.
  • ③ 메뉴에서 '홈 화면에 추가 ➕'를 선택한다.

안드로이드(Android)에서 추가하기

  • 크롬: 킥톡 화면에 뜨는 '설치' 버튼을 한 번 누르면 끝. 배너가 안 보이면 브라우저 메뉴 (⋮) → '앱 설치' 또는 '홈 화면에 추가'를 선택한다.
  • 삼성 인터넷 등: 브라우저 메뉴 (≡ 또는 ⋮)를 열고 '현재 페이지를 홈 화면에 추가'를 고른다.
  • 카카오톡 링크로 들어온 경우: 인앱 브라우저에서는 설치가 안 되므로, '다른 브라우저로 열기'로 크롬·삼성 인터넷에서 연 뒤 추가한다.

킥톡 핵심 기능 활용법

  • 실시간 스코어·경기 시간: 진행 중인 경기의 점수와 현재 몇 분인지 실시간으로 표시.
  • 예상·확정 라인업: 경기 전 예상 라인업, 발표 후 확정 라인업을 실제 포메이션 그림으로 확인.
  • 선수 평점·MVP: 경기가 끝나면 선수별 평점과 팀 평균을 색상 배지로 표시. 직접 별점으로 MVP도 뽑는다.
  • 대진표: 조별리그 순위에 따라 채워지는 토너먼트 대진과 진출팀을 한눈에.
  • 득점 순위: 대회 골든부트 레이스를 실시간 득점 순위로 추적.
  • 응원·게시판: 경기별 응원 메시지와 커뮤니티 게시판에서 다른 팬들과 소통.
  • 하이라이트: 경기 종료 후 주요 장면 하이라이트 링크 제공.

결과를 미리 알기 싫다면 — 스포일러 방지 모드

경기를 나중에 몰아 볼 계획이라면, 헤더의 🙈 스포일러 방지 버튼을 켜 두자. 스코어·결과·진출팀 같은 결과 정보를 화면에서 가려, 라인업과 일정만 보며 김빠지지 않게 즐길 수 있다. (하이라이트 링크는 유지된다.)

로그인하면 더 편하다

구글 계정으로 간단히 로그인하면 닉네임을 설정하고, 응원·평점·게시판 활동을 이어서 관리할 수 있다. 로그인 없이도 경기 정보는 모두 무료로 볼 수 있으니 부담 없이 시작하면 된다.

지금 킥톡(kicktalk.xyz)을 홈 화면에 추가하고, 2026 북중미 월드컵의 모든 경기를 라인업부터 평점까지 한 곳에서 즐겨보자.
