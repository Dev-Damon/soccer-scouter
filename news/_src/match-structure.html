축구 경기 시간은 몇 분? 전·후반·추가시간·연장 완전 정리

"축구 경기는 몇 분이나 하나요?"는 축구를 처음 보는 사람이 가장 많이 던지는 질문이다. 기본은 90분이지만, 하프타임과 추가시간, 토너먼트의 연장전까지 더하면 실제 경기장에 머무는 시간은 두 시간을 훌쩍 넘기기도 한다. 경기 시간의 구성을 정리했다.

기본 — 전반 45분 + 후반 45분

축구 경기는 전반 45분, 후반 45분으로 총 90분이 정규 시간이다. 전반과 후반 사이에는 약 15분의 하프타임(휴식)이 있다. 농구·배구처럼 시간이 멈추지 않고, 심판이 시계를 계속 돌린다는 점이 특징이다.

추가시간(인저리타임) — 흘려보낸 시간 보충

경기 중에는 부상 치료, 선수 교체, 골 세리머니, VAR 판독 등으로 실제 플레이가 멈추는 시간이 생긴다. 이렇게 허비된 시간을 각 하프가 끝날 때 추가시간(스토피지 타임, 인저리타임)으로 보충한다. 전반 종료 시 1~5분, 후반 종료 시 3~10분 정도가 흔하며, 최근에는 실제 지연 시간을 더 엄격히 반영해 추가시간이 길어지는 추세다. "90+4분"은 후반 90분에 더해진 4분째라는 뜻이다.

연장전 — 토너먼트에서만

무승부가 허용되는 조별리그는 90분(+추가시간)으로 끝난다. 하지만 승자를 반드시 가려야 하는 토너먼트에서 90분 뒤에도 동점이면 전·후반 15분씩, 총 30분의 연장전을 치른다. 연장전에도 하프 사이 짧은 휴식이 있고, 연장에서 넣은 골도 당연히 유효하다.

그래도 안 갈리면 — 승부차기

연장 30분까지 치르고도 동점이면 마지막 관문인 승부차기로 넘어간다. 양 팀이 5명씩 페널티킥을 차 더 많이 넣는 팀이 이기고, 그래도 같으면 한 명씩 번갈아 차는 서든데스로 승자를 가린다.

정리 — 상황별 총 경기 시간

  • 조별리그: 90분 + 추가시간(대략 100분 안팎)
  • 토너먼트에서 연장까지: 90분 + 30분 + 추가시간(대략 130분 안팎)
  • 승부차기까지: 위에 더해 킥 대결 시간까지

킥톡에서는 진행 중인 경기의 실시간 경기 시간(분)과 스코어를 그대로 보여준다. 추가시간에 터지는 극적인 결승골까지 놓치지 말고 따라가 보자.
